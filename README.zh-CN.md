# wei-updater

- [ ] 当前目录加载udpate.dat,获取线上路私
- [ ] 自动对比远程服务器上面的版本号

# 示例

- update.dat

url: http://updater.zuiyue.com/

- 服务器目录结构

````
/				                # 根目录
/windows			            # windows 文件存放
/windows/version.dat		    # 版本号对比
/windows/updater.zip		    # 更新文件
/ubuntu				            # ubuntu 更新文件存放
/ubuntu/version.dat             # 版本号对比
/ubuntu/updater.zip		        # 更新文件
````
